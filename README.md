## Overview
This repository provides the complete supplementary materials associated with the article:

> **Ultrasound-Enhanced Data-Driven Modeling for Characterizing Natural Wrist Tremor Dynamics**  
> X. Xue, V. Ganesh, A. Iyer, D. Roque, X. Jiang, and N. Sharma  
> IEEE Journal of Biomedical and Health Informatics (JBHI), 2025.  

The supplementary files expand on the methodology, mathematical derivations, model implementation details, and experimental protocols that support the results reported in the manuscript. These materials are intended to enhance transparency, reproducibility, and accessibility for researchers working in data-driven modeling, ultrasound-based sensing, and tremor dynamics characterization.

---

## Contents

### **1. Supplementary PDF (Full Technical Details)**
The PDF includes:

- **S1. Koopman Operator and Linear Predictor Derivations**  
  Full step-by-step mathematical development of the EDMD-based lifted model and the Koopman operator formulation.

- **S2. Real-Time Model Updating using Recursive Least Squares (RLS)**  
  Complete derivation of adaptive parameter updates, including forgetting factor, stability considerations, and selective update strategy.

- **S3. Ultrasound Speckle-Tracking Displacement Computation**  
  Equations, correlation formulation, displacement extraction, directional conventions, and processing workflow.

- **S4. Detailed Construction of Observable Functions for EDMD**  
  Basis functions, harmonic terms, radial basis functions, and nonlinear feature construction.

- **S5. Quantitative Evaluation Metrics**  
  - NRMSE (time domain)  
  - Dominant frequency extraction  
  - Tremor power ratio  
  - Implementation notes

- **S6. Tremor Event Detection and Validation Metrics**  
  STFT configuration, thresholding logic, precision/recall/F1-score calculations.

- **S7–S9. Additional Analysis and Between-Group Comparisons**  
  - ET vs PD modeling trends  
  - Case 3 vs Case 4 comparisons  
  - Physiological interpretation of eigenvalue-based tremor characteristics
